# tools

Small CLI tools for daily workflows.

## Tools

- [fast-format-x](https://ffx.bfoos.net) (`ffx`) - Format every changed file in parallel across languages.
- [gather-files](https://gf.bfoos.net) (`gf`) - Gather source files with README first, stitch with headers, copy to the clipboard, and report stats.
- [git-new-branch](https://gnb.bfoos.net) (`gnb`) - Create git branches with a username prefix and auto-increment on collisions.
